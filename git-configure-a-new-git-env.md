# Configuration for a new git environment

## User
```shell
git config --global user.name "John Doe"
```

## Email
```shell
git config --global user.email johndoe@example.com
```

## Editor
```shell
git config --global core.editor vim
```

## Check the current configuration
```shell
git config --list
```

## Timezone
```shell
tzselect
```
